🍀 B2B Loterias

Instruções para acessar de outro computador (Modo App - Sem barra do Chrome)

1

No computador PRINCIPAL (servidor)

Dê duplo clique no atalho B2B Loterias REDE na Área de Trabalho. NÃO feche a janela preta que aparece.

2

Envie este arquivo para o OUTRO computador

Copie o arquivo abaixo para o outro computador via pen drive, WhatsApp, e-mail ou qualquer meio:

ABRIR_B2B_REMOTO.bat
Basta dar duplo clique neste arquivo no outro computador.
O sistema abrirá em tela cheia, sem barra de navegação, igual ao seu computador.
3

Pronto!

O B2B Loterias abrirá no modo aplicativo (sem barra do Chrome), exatamente como aparece na sua Área de Trabalho.

⚠️ IMPORTANTE

• Os dois computadores devem estar na mesma rede Wi-Fi.
• A janela preta do servidor deve ficar aberta no computador principal.
• Se o Windows pedir para liberar o Firewall, clique em "Permitir acesso".
• Se o IP mudar, edite o arquivo ABRIR_B2B_REMOTO.bat e troque o IP na linha set SERVIDOR=.
